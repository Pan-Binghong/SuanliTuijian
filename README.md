# 算力卡推荐工具v2

这是一个基于 Vue 3 的算力卡推荐应用程序。

## 项目设置

### 前置条件

- Node.js（与 Vue 3 兼容的版本）
- npm 或 yarn

### 安装

1. 克隆仓库：
   ```
   git clone [仓库地址]
   cd suanlituijian
   ```

2. 安装依赖：
   ```
   npm install
   ```


## 开发

启动开发服务器：
```
npm run dev
```


## 项目结构

- `index.html`：主 HTML 文件
- `src/`：源代码文件
  - `main.js`：应用程序入口点
- `public/`：公共资源
  - `傻猫.png`：网站图标

## 使用的技术

- Vue 3
- Vue Router
- Vite

## 配置

项目使用 Vite 作为构建工具。配置可以在以下文件中找到：
- `vite.config.js`
